第3章 神经网络

Q：为什么批处理可以缩短处理时间？

A：因为大多数处理数值运算的库都进行了能够高效处理大型数组运算的最优化。并且，在神经网络的运算中，当数据传送成为瓶颈时，批处理可以减轻数据总线的负荷（相对于数据读入，可以将更多时间用在计算上）。

特征量是指可以从输入数据中准确地提取本质数据的转换器，通常为向量的形式。在CV领域，常用的特征量包括SIFT、SURF和HOG等。使用这些特征量将图像数据转换为向量，然后对转换后的向量使用机器学习中的分类器进行学习。但是使用的特征量仍然是由人设计的，对于不同的问题必须使用合适的特征量。而在神经网络中，连图像中包含的重要特征量也都是由机器来学习的。深度学习也称端到端学习。端到端是指从一段到另一端的意思，也就是从原属数据中获得目标结果的意思。

Q：为什么将数据分为训练数据和测试数据？

A：追求模型的泛化能力（处理未观察过的数据的能力）。

过拟合：只对某个数据集过度拟合的状态

第4章 神经网络的学习

4.2 损失函数

损失函数是表示神经网络性能的恶劣程度的指标，即当前的神经网络对训练数据在多大程度上不拟合。

mini-batch学习：如果遇到大数据，以全部数据为对象计算损失函数是不现实的，因此我们从全部数据中选出一部分，作为全部数据的近似。神经网络的学习也是从训练数据中选出一批数据（称为mini-batch），然后对每个mini-batch进行学习。

Q：为何要设定损失函数？

A：在学习时不能将识别精度作为指标，因为如果这样，参数的导数在大多数地方都会变为0。例如网络正确识别出了100笔训练数据中的32笔，识别精度为32%，即使稍微改变参数的值，识别精度也仍将保持在32%，不会出现像32.0123%这样连续变化。而如果把损失函数作为指标，则当前损失函数的值可以表示为0.9254这样的值。

4.4 梯度

梯度：由全部变量的偏导数汇总而成的向量

第5章 误差反向传播法

上一章介绍了通过数值微分计算损失函数关于权重参数的梯度，这种方式简单但费时，本章介绍能够高效计算的误差反向传播法

正向传播是从计算图出发点到结束点的传播

在确认误差反向传播法的实现是否正确时，时需要用到数值微分的。请确认数值微分求出的梯度结果和误差反向传播法求出的结果是否一致（非常相近）的操作称为梯度确认。

第6章 与学习相关的技巧

6.1 参数的更新

随机梯度下降法 SGD：朝着当前所在位置的坡度最大的方向前进

Momentum：一直在同一方向上受力，所以朝同一个方向会有一定的加速

AdaGrad：随着学习的进行，使学习率逐渐减小（学习率衰减），针对一个一个的参数，赋予其定制的值。参数的元素中变动较大（被大幅更新）的元素的学习率将变小。

Adam：融合Momentum和AdaGrad

6.2 权重的初始值

Q：为什么不能将权重初始值设成一样的值？

A：因为在误差反向传播法中，所有的权重值都会进行相同的更新，这使得神经网络拥有许多不同的权重的意义丧失了。

当激活函数使用ReLU时，权重初始值使用He初始值（标准差为$\sqrt{\frac{2}{n}}$的高斯分布）：w = np.random.randn(node_num, node_num)*np.sqrt(2/node_num)

当激活函数为sigmoid或tanh等S型曲线函数时，初始值使用Xavier初始值（标准差为$\sqrt{\frac{1}{n}}$的高斯分布）：w = np.random.randn(node_num, node_num)/np.sqrt(node_num)

6.3 Batch Normalization

以进行学习时的mini-batch为单位进行正规化。具体而言，就是进行使数据分布的均值为0、方差为1的正规化。通过将这个处理插入到激活函数的前面（或者后面），可以减小数据分布的偏向。

思路：调整各层的激活值分布使其拥有适当的广度

有点：可以使学习快速进行；不那么依赖初始值；抑制过拟合

6.4 正则化

发生过拟合的原因：模型拥有大量参数、表现力强；训练数据少

权值衰减：通过在学习的过程中对大的权重进行惩罚来抑制过拟合。为损失函数加上$\frac{1}{2}\lambda W^2$，在求权重梯度的计算中，要为之前的误差反向传播法的结果加上正则化项的导数$\lambda W$。但是如果网络模型变得很复杂，只用权值衰减就难以应对了。

Dropout：在学习过程中随机删除神经元，被删除的神经元不再进行信号的传递。可以将Dropout理解为通过在学习过程中随机删除神经元，从而每一次都让不同的模型进行学习，并且在推理时通过对神经元的输出乘以删除比例，可以获得模型的平均值。

6.5 超参数的验证

Q：为什么不能用测试数据评估超参数的性能？

A：这样会导致超参数的值被调整为只拟合测试数据，因此需要使用验证数据来评估超参数的好坏

第7章 卷积神经网络

一般来说，池化的窗口大小会和步幅设定成相同的值

基于im2col的展开：im2col会把输入数据展开以适合滤波器。具体来说，将应用滤波器的区域（3维方块）横向展开为一列。之后就只需要将卷积层的滤波器纵向展开为一列，并计算两个矩阵的乘积，最后reshape为输出数据的大小即可。

池化层也使用im2col展开输入数据，不过在通道方向上是独立的，这一点和卷积层不同。具体来讲，池化的应用区域按通道单独展开。展开之后，只需对展开的矩阵求各行的最大值，并转换为合适的形状即可。

叠加小型滤波器来加深网络的好处是可以减少参数量。例如，一次$5 \times 5$的卷积运算的区域可以由两次$3 \times 3$的卷积运算抵充，而前者参数量为25（$5 \times 5$），后者参数量为18（$2 \times 3 \times 3$）。
